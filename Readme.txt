HireMe - Recruitment Application


Project Overview

HireMe is a modern recruitment application built using Spring Boot, React.js, Spring Security 6, and JWT (JSON Web Tokens) for Identity and Access Management (IAM). It streamlines the hiring process, making it easy for employers to post job openings and for job seekers to apply. The application provides secure and efficient user authentication and access control.

Features : 

    User Authentication: Utilizes Spring Security 6 for secure user authentication and JWT for token-based IAM.
    Role-Based Access Control: Implements role-based access control to manage user permissions.
    Job Postings: Employers can post job openings, including details, requirements, and application instructions.
    Job Applications: Job seekers can apply for positions, attach their resumes, and track their application status.
    User Profiles: Users can create and manage their profiles, including contact information, education, work experience, and skills.
    Notifications: Provides notifications for job application updates and new job postings.

Technologies Used

    Spring Boot: A Java-based framework for the backend.
    React.js: A JavaScript library for building the user interface.
    Spring Security 6: Ensures secure user authentication and access control.
    JWT (JSON Web Tokens): Used for IAM and user token management.
    Database: MySQL
    Other libraries/tools:Hibernate, lomboc, React, Spring boot/cloud, Jpa, Spring Security 6 ...
